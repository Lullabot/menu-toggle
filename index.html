Menu Toggle - Javascript to help create custom collapsible menus

This code can help you create a custom hamburger, falafel, kebob, and other expanding menus. This library still requires styling to make it look and animate as desired, but below are a few example implementations.

Features:

  • Manages state of all 'menu toggles' and can manage dependencies between them (e.g. can be expanded at the same time or should close all others)
  • Adds open and shut functions to collapsible elements
  • Hiding the menu-toggle button (e.g. the hamburger) will open the collapsible container and disable the collapsible functionality so the collapsible behavior can be toggled from CSS.
  • Adds keyboard navigation to menu
  • Manages accessibilty properties
  • Can have a no-js implementation (if JS is loaded it will rip out no-js markup since it's not as accessible)

      <input id="desktop-burger-toggle" class="menu-toggle u-element-invisible" type="checkbox" aria-controls="desktop-burger-menu-container">
      <label class="menu-toggle__button" for="desktop-burger-toggle" data-controls="desktop-burger-menu-container">
        Menu icon or Label Text
        <span class="menu-toggle-assistive-text u-element-invisible">Toggle menu visibility</span>
      </label>
      <div class="menu-toggle__toggleable">
        <div class="menu-toggle__toggleable-content-wrapper">
          Content in Collapsible Container
        </div>
      </div>
